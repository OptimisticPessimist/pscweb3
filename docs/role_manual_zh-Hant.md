# PSCWEB3 使用者手冊
## 戲劇與影視製作專案管理系統

本系統旨在簡化戲劇演出和影視製作專案的管理。
包含劇本管理、行程安排、考勤追蹤、演職人員管理等製作所需的各項功能。

## 1. 關於角色

系統有三種角色。請確認您的角色。

### 👑 擁有者 (Owner)
**戲劇:** 社長、導演、製作主管  
**影視:** 導演、製片人、製作總監

- 建立和刪除專案
- 邀請成員和管理權限
- 設定 Discord 通知
- 擁有所有功能的存取權限

### ✏️ 編輯者 (Editor)
**戲劇:** 導演助理、舞台監督、各部門負責人  
**影視:** 副導演、攝影指導、部門主管

- 上傳和編輯劇本
- 建立和編輯行程
- 向未回覆者發送考勤提醒
- 管理場景表
- 設定選角

### 👀 檢視者 (Viewer)
**戲劇:** 演員、工作人員、顧問  
**影視:** 表演者、工作人員、客戶

- 查看行程
- 閱讀劇本
- 回覆考勤
- 查看個人行程

## 2. 入門指南（所有使用者）

### 🔐 如何登入
1. 存取系統 URL。
2. 點擊 **「使用 Discord 登入」** 按鈕。
3. 使用您的 Discord 帳號進行身分驗證即可自動登入。

> 💡 如果您沒有 Discord 帳號，請先在 [Discord](https://discord.com/) 建立免費帳號。

### 🏠 儀表板
登入後，您將看到儀表板：
- 您參與的專案清單
- 即將到來的行程
- 各項功能的連結

### 🌐 語言切換
使用右上角的按鈕切換顯示語言：
- 日本語 / English / 한국어 / 简体中文 / 繁體中文

## 3. 檢視者手冊

主要供演員和工作人員使用的功能。

### 📅 查看行程
查看排練和拍攝日期。

1. 從選單中選擇 **「行程」**。
2. 行程以行事曆形式顯示。
3. 點擊日期查看詳情（地點、場景、參與者等）。

### 📆 我的行程
僅查看您參與的排練/拍攝。

1. 從選單中選擇 **「我的行程」**。
2. 顯示您被選角的場景的排練。

### ✅ 回覆考勤
回覆排練或拍攝的考勤。

**透過 Discord:**
1. 您將在 Discord 上收到通知。
2. 只需按「出席」或「缺席」按鈕即可完成。

**在系統內:**
1. 從選單中選擇 **「考勤」**。
2. 選擇活動並回覆考勤。

### 📖 閱讀劇本
隨時從手機或電腦存取最新劇本。

1. 從選單中選擇 **「劇本」**。
2. 從清單中選擇要閱讀的劇本。
3. 劇本以易讀的直排格式顯示。

### 📊 查看場景表
查看每個場景中出現的角色。

1. 在劇本詳情頁面，選擇 **「場景表」** 標籤。
2. 列出每個場景中出現的角色。

## 4. 編輯者手冊

供導演和製作人員等專案管理者使用的功能。

### 📝 上傳劇本
上傳 Fountain 格式的劇本檔案。

1. 從選單中選擇 **「劇本」** → **「上傳」**。
2. 選擇檔案（或拖放）。
3. 確認標題並點擊 **「上傳」**。

> 💡 **什麼是 Fountain 格式？**  
> 一種用於編寫劇本的簡單文字格式。  
> 瞭解更多請造訪 [fountain.io](https://fountain.io/)。

### 🗓 建立行程
建立排練或拍攝行程。

1. 從選單中選擇 **「行程」**。
2. 點擊 **「新建」** 按鈕。
3. 輸入以下內容並儲存：
   - 日期和時間
   - 地點
   - 目標場景（可多選）
   - 備註

### 📊 編輯場景表
設定每個場景中出現的角色。

1. 在劇本詳情頁面，選擇 **「場景表」** 標籤。
2. 在每個場景列中勾選出現的角色。
3. 變更自動儲存。

### 🎭 設定選角
將角色與演員關聯。

1. 從選單中選擇 **「選角」**。
2. 從清單中選擇要設定的角色。
3. 選擇分配的演員。
4. 也支援雙重選角（同一角色多個演員）。

### 👥 工作人員管理
設定工作人員角色。

1. 從選單中選擇 **「工作人員」**。
2. 在成員清單中設定每個工作人員的分配角色。

### 📢 發送考勤提醒
向未回覆的成員發送提醒。

1. 從選單中選擇 **「考勤」**。
2. 選擇目標活動。
3. 點擊 **「發送提醒」** 按鈕。
4. 透過 Discord 向未回覆者發送帶有提及的提醒。

## 5. 擁有者手冊

供整個專案管理者使用的功能。

### 🚀 建立專案
建立新的演出或拍攝專案。

1. 在儀表板上點擊 **「建立新專案」**。
2. 輸入專案名稱（演出標題等）。
3. 輸入說明（選用）並點擊 **「建立」**。

### 📩 邀請成員
邀請成員加入專案。

1. 開啟專案 **「設定」**。
2. 點擊 **「建立邀請連結」** 按鈕。
3. 複製顯示的 URL 並透過 LINE 或 Discord 分享。
4. 當成員透過連結登入時，他們會自動加入專案。

### ⚙️ 變更成員權限
變更成員的角色（擁有者/編輯者/檢視者）。

1. 開啟專案的 **「設定」** → **「成員管理」**。
2. 選擇要變更的成員的 **「角色」**。
3. 變更立即生效。

### 🔔 Discord 通知設定
將專案通知發送到 Discord 的設定。

#### 🤖 邀請 Discord Bot（僅限首次）
要發送帶有按鈕的考勤確認訊息，您需要將 Discord Bot 邀請到您的伺服器。

> ⚠️ **此操作需要 Discord 伺服器管理員權限。**

1. 存取以下 URL：
   
   👉 **[邀請 Discord Bot](https://discord.com/oauth2/authorize?client_id=1447907388337422398&permissions=2048&scope=bot)**
2. 選擇要邀請 Bot 的 Discord 伺服器。
3. 點擊 **「授權」**。
4. 確認 Bot 已加入伺服器。

> 💡 **所需權限**: 只需要「發送訊息」權限。

#### Webhook URL（一般通知）
接收專案更新等通知。

1. 開啟要接收通知的 Discord 頻道的 **「頻道設定」**。
2. 選擇 **「整合」** → **「Webhook」**。
3. 建立 **「新 Webhook」** 並複製 URL。
4. 將其貼上到系統設定中的 **「Discord Webhook URL」** 欄位。

#### Webhook URL（劇本通知）
劇本上傳通知可以發送到單獨的頻道（選用）。

#### 頻道 ID（考勤）
用於發送帶有按鈕的考勤確認訊息的設定。

**如何取得頻道 ID:**
1. 開啟 Discord **「使用者設定」** → **「進階」**。
2. 開啟 **「開發者模式」**。
3. **右鍵點擊** 目標頻道 → 選擇 **「複製頻道 ID」**。
4. 將其貼上到系統設定中的 **「Discord 頻道 ID」** 欄位。
