## 1. 关于角色

系统有三种角色。请确认您的角色。

### 👑 所有者 (Owner)
**戏剧:** 社长、导演、制作主管  
**影视:** 导演、制片人、制作总监

- 创建和删除项目
- 邀请成员和管理权限
- 配置 Discord 通知
- 拥有所有功能的访问权限

### ✏️ 编辑者 (Editor)
**戏剧:** 导演助理、舞台监督、各部门负责人  
**影视:** 副导演、摄影指导、部门主管

- 上传和编辑剧本
- 创建和编辑日程
- 向未回复者发送考勤提醒
- 管理场景表
- 配置选角

### 👀 查看者 (Viewer)
**戏剧:** 演员、工作人员、顾问  
**影视:** 表演者、工作人员、客户

- 查看日程
- 阅读剧本
- 回复考勤
- 查看个人日程

## 2. 入门指南（所有用户）

### 🔐 如何登录
1. 访问系统 URL。
2. 点击 **"使用 Discord 登录"** 按钮。
3. 使用您的 Discord 账号进行身份验证即可自动登录。

> 💡 如果您没有 Discord 账号，请先在 [Discord](https://discord.com/) 创建免费账号。

### 🏠 仪表板
登录后，您将看到仪表板：
- 您参与的项目列表
- 即将到来的日程
- 各项功能的链接

> 💡 **关于项目创建数量的限制**
> 通常，一个用户作为所有者最多可以创建 **2个** 项目（私有）。
> 但是，**包含“公开设置”剧本的项目不在此限制范围内**，可以无限创建。
> 多多公开作品，扩展活动范围吧！

### 🌐 语言切换
使用右上角的按钮切换显示语言：
- 日本語 / English / 한국어 / 简体中文 / 繁體中文

## 3. 🎫 票务预约 (一般观众)

本部分针对前来观看演出的 **一般观众**。  
※ 项目成员（演职人员）请参阅后续部分。

### 票务预约方法

1. 访问通过 SNS 或官方网站分享的 **预约页面 URL**。
2. 输入以下信息：
   - **姓名**: 预约者姓名
   - **电子邮件**: 接收确认邮件的地址
   - **预约张数**: 需要的票数（1张或多张）
   - **推荐人**: （可选）如果您认识演职人员，请选择
3. 点击 **“预约”** 按钮。
4. 您将收到确认邮件 - 请查看详情。

> 💡 **预约完成后**  
> - 确认邮件包含演出日期、地点和预约详情。
> - 还包含添加到 Google 日历的链接。
> - 如需取消，请使用邮件中的取消链接。

> 🌐 **多语言支持**  
> 预约页面支持 5 种语言（日语、英语、韩语、简体中文、繁体中文）。  
> 它会根据您的浏览器语言设置自动显示。

## 4. 查看者手册

主要供演员和工作人员使用的功能。

### 📅 查看日程
查看排练和拍摄日期。

1. 从菜单中选择 **"日程"**。
2. 日程以日历形式显示。
3. 点击日期查看详情（地点、场景、参与者等）。

### 📆 我的日程
仅查看您参与的排练/拍摄。

1. 从菜单中选择 **"我的日程"**。
2. 显示您被选角的场景的排练。

### ✅ 回复考勤
回复排练或拍摄的考勤。

**通过 Discord:**
1. 您将在 Discord 上收到通知。
2. 只需按"出席"或"缺席"按钮即可完成。

**在系统内:**
1. 从菜单中选择 **"考勤"**。
2. 选择活动并回复考勤。

### 📖 阅读剧本
随时从手机或电脑访问最新剧本。

1. 从菜单中选择 **"剧本"**。
2. 从列表中选择要阅读的剧本。
3. 剧本以易读的竖排格式显示。

### 🌍 浏览公开剧本
可以阅读其他用户公开的剧本。

1. 从仪表板或菜单选择 **“公开剧本”**。
2. 选择感兴趣的剧本进行浏览。
3. 喜欢的剧本也可以 **导入 (Import)** 到自己的项目中（详见后文）。

### 📊 查看场景表
查看每个场景中出现的角色。

1. 在剧本详情页面，选择 **"场景表"** 标签。
2. 列出每个场景中出现的角色。

## 5. 编辑者手册

供导演和制作人员等项目管理者使用的功能。

### 📝 上传剧本
上传 Fountain 格式的剧本文件。

1. 从菜单中选择 **"剧本"** → **"上传"**。
2. 选择文件（或拖放）。
3. 确认标题并点击 **"上传"**。

> 💡 **什么是 Fountain 格式？**  
> 一种用于编写剧本的简单文本格式。  
> 了解更多请访问 [fountain.io](https://fountain.io/)。
>
> **🇯🇵 日语 Fountain 扩展语法**:
> 本系统支持为了方便编写日语剧本的扩展语法。
> 1. **一行台词**: 以 `@` 开头（如 `@角色名 台词`），可以在一行内写完角色和台词。
> 2. **强制场景标题**: 以 `.` 开头（如 `.1 第一幕` 或 `.2 第一场`），会被识别为场景标题。
> 3. **保留全角/半角空格**: 行首输入空格会被识别为动作说明（舞台提示），并保留缩进。

### 📝 输入剧本信息
上传时可以设置以下信息：
- **作者名**: 剧本作者的名字（可选）。
- **公开设置**: 勾选“公开”后，其他用户可以浏览或导入。
    - ※ 公开后将不计入项目创建数量限制。

### 🗓 创建日程
创建排练或拍摄日程。

1. 从菜单中选择 **"日程"**。
2. 点击 **"新建"** 按钮。
3. 输入以下内容并保存：
   - 日期和时间
   - 地点
   - 目标场景（可多选）
   - 备注

### 🚩 里程碑设置
设置重要截止日期和演出日期（里程碑）。

1. 从项目设置中的 **“里程碑设置”** 添加。
2. 设置标题、日期、颜色等，然后保存。
3. 显示在日程上；可以自动创建考勤确认事件。
4. 设置 **预约容量** 会生成票务预约页面链接。
   - 将生成的链接与演出宣传一起在 SNS 上分享。
   - 观众可以通过输入姓名、邮箱、张数等来预约。

### 🎫 里程碑别预约列表
查看每场演出（里程碑）的预约状况并管理观众考勤。

1. 打开 **“里程碑设置”** 在项目设置中。
2. 点击设置了预约容量的里程碑的 **“预约页面”** 链接。
3. 预约列表显示以下信息：
   - 预约者姓名和电子邮件地址
   - 预约张数
   - 推荐人（如果是演职人员）
   - 预约日期和时间
4. 使用 **考勤切换** 记录观众的考勤状况。
   - 开启切换会标记为“已出席”。
   - 列表清楚地显示已出席与未出席。

> 💡 **多语言支持**  
> 预约页面支持 5 种语言（日语、英语、韩语、简体中文、繁體中文）。  
> 它会根据观众的浏览器设置自动显示。

### 📊 编辑场景表
设置每个场景中出现的角色。

1. 在剧本详情页面，选择 **"场景表"** 标签。
2. 在每个场景行中勾选出现的角色。
3. 更改自动保存。

### 🎭 配置选角
将角色与演员关联。

1. 从菜单中选择 **"选角"**。
2. 从列表中选择要配置的角色。
3. 选择分配的演员。
4. 也支持双重选角（同一角色多个演员）。

### 👥 工作人员管理
设置工作人员角色。

1. 从菜单中选择 **"工作人员"**。
2. 在成员列表中设置每个工作人员的分配角色。

### 📢 发送考勤提醒
向未回复的成员发送提醒。

1. 从菜单中选择 **"考勤"**。
2. 选择目标活动。
3. 点击 **"发送提醒"** 按钮。
4. 通过 Discord 向未回复者发送带有提及的提醒。

## 6. 所有者手册

供整个项目管理者使用的功能。

### 🚀 创建项目
创建新的演出或拍摄项目。

1. 在仪表板上点击 **"创建新项目"**。
2. 输入项目名称（演出标题等）。
3. 输入描述（可选）并点击 **"创建"**。

### 📥 导入剧本
基于公开的剧本创建新项目。

1. 打开 **“公开剧本”** 页面。
2. 点击想使用的剧本的 **“导入”** 按钮。
3. 输入新项目名称并创建。
   - 导入的剧本将作为 **“私有”** 设置（副本）保存在您的项目中。
   - 再次设置为公开，可以享受不计入限制的优惠。

### 📩 邀请成员
邀请成员加入项目。

1. 打开项目 **"设置"**。
2. 点击 **"创建邀请链接"** 按钮。
3. 复制显示的 URL 并通过 LINE 或 Discord 分享。
4. 当成员通过链接登录时，他们会自动加入项目。

### ⚙️ 更改成员权限
更改成员的角色（所有者/编辑者/查看者）。

1. 打开项目的 **"设置"** → **"成员管理"**。
2. 选择要更改的成员的 **"角色"**。
3. 更改立即生效。

### 🔔 Discord 通知设置
将项目通知发送到 Discord 的设置。

#### 🤖 邀请 Discord Bot（仅限首次）
要发送带有按钮的考勤确认消息，您需要将 Discord Bot 邀请到您的服务器。

> ⚠️ **此操作需要 Discord 服务器管理员权限。**

1. 访问以下 URL：
   
   👉 **[邀请 Discord Bot](https://discord.com/oauth2/authorize?client_id=1447907388337422398&permissions=2048&scope=bot)**
2. 选择要邀请 Bot 的 Discord 服务器。
3. 点击 **"授权"**。
4. 确认 Bot 已加入服务器。

> 💡 **所需权限**: 只需要"发送消息"权限。

#### Webhook URL（常规通知）
接收项目更新等通知。

1. 打开要接收通知的 Discord 频道的 **"频道设置"**。
2. 选择 **"集成"** → **"Webhook"**。
3. 创建 **"新 Webhook"** 并复制 URL。
4. 将其粘贴到系统设置中的 **"Discord Webhook URL"** 字段。

#### Webhook URL（剧本通知）
剧本上传通知可以发送到单独的频道（可选）。

#### 频道 ID（考勤）
用于发送带有按钮的考勤确认消息的设置。

**如何获取频道 ID:**
1. 打开 Discord **"用户设置"** → **"高级"**。
2. 打开 **"开发者模式"**。
3. **右键点击** 目标频道 → 选择 **"复制频道 ID"**。
4. 将其粘贴到系统设置中的 **"Discord 频道 ID"** 字段。
