# 脚本ビューアーの表示順序変更 修正確認 (Walkthrough)

脚本ビューアーにおいて、あらすじ（シーン番号0）を登場人物リストの前に表示するように順序を変更しました。

## 変更内容

### 1. 多言語対応 (`frontend/src/locales/`)
各言語の翻訳ファイルに `script.synopsis` キーを追加しました。
- 日本語: 「あらすじ」
- 英語: "Synopsis"
- 韓国語: "줄거리"
- 簡体字・繁体字: "梗概"

### 2. コンポーネントの修正 (`frontend/src/features/scripts/pages/`)
以下のファイルにおいて、表示順序を制御するロジックを実装しました。
- `ScriptDetailPage.tsx`
- `PublicScriptPage.tsx`

**実装詳細:**
- 全てのシーンリストから `scene_number === 0` のものを `synopsisScene` として抽出しました。
- 残りのシーンを `mainScenes` としました。
- `synopsisScene` が存在する場合、メタデータセクションの後、登場人物セクションの前に独立したセクションとしてレンダリングします。
- 脚本内容セクションでは `mainScenes` のみを表示するようにし、あらすじが重複して表示されないようにしました。

## 検証結果
- あらすじが存在する場合、正しく登場人物の前に表示されます。
- あらすじが存在しない場合、セクション自体が表示されず、レイアウトに影響を与えません。
- メインの脚本内容からはあらすじが除外されていることを確認しました。
