# 役割別利用マニュアルの実装 (Walkthrough)

## 概要
従来のシステム権限（管理者/編集者/閲覧者）に基づくマニュアルを、演劇制作の現場で使われる役職（制作/演出/脚本/演出助手・舞台監督/役者/テクニカルスタッフ）に基づいた構成に刷新しました。

## 変更内容

### 1. タブ切り替えUIの導入
`ManualPage.tsx` を刷新し、Headless UI を用いたタブ切り替えインターフェースを導入しました。これにより、ユーザーは自分の役職に関連する項目だけを素早く確認できるようになりました。

### 2. コンテンツの再構成
- **共通セクション**: 全ユーザーが必要な「ログイン方法」「ダッシュボードの使い方」「言語切り替え」などを「はじめに」として常に表示。
- **役割別セクション**: 6つの役職ごとに、推奨される権限レベルと、その役職が主に使用する機能のチェックリストを表示。

### 3. 多言語対応の強化
- **5言語サポート**: 日本語、英語、韓国語、簡体字、繁体字のすべての言語で、役職別のコンテンツへの切り替えをサポート。
- **翻訳キーの追加**: `translation.json` に各役職の名前やUIラベルを追加。

### 4. URLパラメータによる直接リンク
`/manual?role=cast` のようにURLパラメータを指定することで、最初から特定の役職のタブが開いた状態で表示される機能を実装しました。これにより、各画面からの「マニュアルを表示」リンクをより適切に設定できるようになります。

### 5. 推奨権限バッジの表示
各役職に対して、システム上どの権限（管理者/編集者/閲覧者）を付与すべきかのガイドラインをバッジ形式で表示するようにしました。

## ファイル構造の変更
保守性を高めるため、マニュアルコンテンツを以下の構造に分割しました。
- `frontend/src/pages/manualContent/common/`: 全共通セクション
- `frontend/src/pages/manualContent/roles/[role_name]/`: 役割別セクション

## 確認方法
1. マニュアルページ（`/manual`）にアクセスします。
2. 「はじめに」の内容が表示されていることを確認します。
3. 役職タブ（制作、演出など）をクリックし、それぞれのコンテンツと「推奨権限」バッジが表示されることを確認します。
4. 言語を切り替えた際、タブのラベルとコンテンツが適切に翻訳されることを確認します。
5. URLに `?role=director` を付けてアクセスし、最初から「演出」タブが開いていることを確認します。
