# Walkthrough - Synopsis Formatting & Cover Page Split

あらすじの表示スタイルを改善し、表紙（タイトル・メタデータ）と本文（あらすじ・第一章以降）を別ページに分ける実装を行いました。

## Changes

### 1. PDF生成ロジックのカスタマイズ
`playscript` ライブラリの内部クラスを使用せず、`backend/src/services/pdf_generator.py` 内にカスタム `PageMan` クラスと生成関数を実装しました。これにより、標準機能では不可能な細かいレイアウト制御が可能になりました。

### 2. 表紙ページと改ページ
- **Author（著者名・メタデータ）** の描画終了後に、強制的に `force_page_break()` を実行するようにしました。
- これにより、タイトルページ（表紙）には「タイトル」「著者名」「その他メタデータ」のみが表示され、次のページから「あらすじ」や「本文」が始まります。

### 3. あらすじのカスタムスタイル
- Header 1 (`# `) が **"Synopsis"** または **"あらすじ"** の場合、そのセクション内を「あらすじモード」として扱います。
- あらすじモード内のテキストは `draw_synopsis_text` メソッドで描画され、**インデント幅を調整**（通常テキストより浅く、セリフより深くなど）し、本文と区別が付くようにしました。
