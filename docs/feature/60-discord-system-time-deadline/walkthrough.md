# Walkthrough - 日程調整などのDiscord期限表示対応

日程調整（Schedule Poll）および出欠確認（Attendance）のDiscord通知において、回答期限を各ユーザーのローカル時刻に合わせて表示するよう改修を行いました。

## 変更内容

### 1. 日程調整の新規作成通知
- `backend/src/services/schedule_poll_service.py` の `create_poll` メソッドを修正。
- 期限 (`deadline`) が設定されている場合、メッセージに `**回答期限:** <t:{timestamp}:f>` を追加しました。

### 2. 日程調整のリマインド通知
- `backend/src/services/schedule_poll_service.py` の `send_reminder` メソッドを修正。
- 通知メッセージに `**回答期限:** <t:{timestamp}:f>` を含めるようにしました。

### 3. 出欠確認のリマインド通知
- `backend/src/services/attendance_tasks.py` の `_send_reminder` メソッドを修正。
- 既存の相対時間表記 `(<t:{timestamp}:R>)` を削除し、日時のみの表示に統一しました。

## 確認事項
- [x] 日程調整作成時に期限が表示されること
- [x] 日程調整リマインド時に期限が表示されること
- [x] 出欠確認リマインドから相対時間が消えていること
- [x] 既存のユニットテストが通過すること
