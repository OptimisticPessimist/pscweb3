# 日程調整カレンダー シーン絞り込み機能 修正内容の確認

## 概要
以前の実装で漏れていた「カレンダービューでのシーン絞り込み」機能を追加し、あわせて候補日作成時のデフォルト時間帯をユーザーの要望に合わせて変更しました。

## 変更内容
### フロントエンド
- **カレンダーコンポーネント (`SchedulePollCalendar.tsx`)**:
  - シーン一覧を自動抽出して絞り込みプルダウンを表示するUIを追加。
  - 選択されたシーンの状態（可能/リーチ/不可）をカレンダー上の色とタイトルに反映。
  - 選択外のイベントを半透明（opacity-40）にすることで、対象日を一目で判別可能に。
  - 詳細パネル内で選択中のシーンを強調表示。
- **日程調整作成ページ (`SchedulePollCreatePage.tsx`)**:
  - デフォルトの開始時間を `22:00`、終了時間を `24:00` に変更。

## 検証結果
### 動作確認内容
1. **シーン絞り込み**:
   - プルダウンからシーンを選択すると、カレンダーのイベントが即座にフィルタリングされることを確認。
   - 選択したシーンが「可能」な日は鮮やかな緑色、「リーチ」な日は黄色、それ以外は薄いグレーになることを確認。
2. **デフォルト時間帯**:
   - 新規作成画面を開いた際、最初の空枠が `22:00 - 24:00` になっていることを確認。
   - 「追加」ボタンを押した際も `22:00 - 24:00` がセットされることを確認。
