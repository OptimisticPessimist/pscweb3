# 日程調整カレンダー 修正内容の確認

## 概要
シーン絞り込みプルダウンが正しく機能しなかった問題、および候補日程の終了時間に「24:00」が指定できなかった問題への修正を行いました。

## 変更内容
### フロントエンド
- **カレンダーコンポーネント (`SchedulePollCalendar.tsx`)**:
  - シーン選択プルダウンが「すべてのシーンを表示」としか表示されず、選択もできなかった問題を修正しました。
  - 原因はCSSの `group` クラスや `z-index` の競合により、スクロールやクリックが要素に到達していなかった事です。`z-index` を調整し、アイコンの配置を改善しました。
  - ルーティングが `main` にマージされた際に欠落していた `ChevronLeft` アイコンのインポートを修正しました。
- **日程調整作成ページ (`SchedulePollCreatePage.tsx`)**:
  - デフォルトの終了時間を `24:00`（ブラウザによってはバリデーションエラーになる）から `23:59` に修正しました。これにより、一日の最後までの枠をスムーズに作成できます。

## 検証結果
### 動作確認内容
1. **シーン絞り込み**:
   - プルダウンをクリックすると、シーンリストが正しく表示されることを確認。
   - シーンを選択すると、カレンダーのフィルタリングが期待通り機能することを確認。
2. **デフォルト時間帯**:
   - 新規作成画面および枠の追加時、デフォルトが `22:00 - 23:59` になっていることを確認。
