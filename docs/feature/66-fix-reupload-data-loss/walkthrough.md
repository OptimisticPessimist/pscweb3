# 修正内容の確認: 脚本再アップロード後の日程調整シーン表示不具合の修正

脚本を再アップロードした際に、日程調整のカレンダー表示や候補日時詳細でシーンが表示されなくなる、あるいは順序が不自然になる不具合を修正しました。

## 修正内容

### Backend

#### [SchedulePollService](file:///f:/src/PythonProject/pscweb3-1/backend/src/services/schedule_poll_service.py)
- **未配役キャラクターの無視**: シーンに登場するキャラクターのうち、誰も配役（`CharacterCasting`）されていない場合は、そのキャラクターを可用性チェックの対象から外すようにしました。これにより、脚本更新で新しいキャラクターが増えたり名前が変わったりしても、シーンが「稽古不可」として隠されることがなくなります。
- **キャラクター不在シーンの表示**: ト書きのみのシーンなど、キャラクターが抽出されなかったシーンも「稽古可能」として分析結果に含めるようにしました。
- **ソート順の修正**: シーンの一覧および分析結果のソートを `act_number`（幕）と `scene_number` の組み合わせで行うように修正しました。

## 検証結果
