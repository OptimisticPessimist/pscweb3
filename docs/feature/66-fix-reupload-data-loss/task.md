# 脚本再アップロード時のデータ紐付け復元

## 課題
脚本を再アップロード（更新）すると、以下のデータが削除または紐付けが解除されてしまう。
1. **配役情報 (CharacterCasting)**: 登場人物（Character）が削除・再作成されるため、ユーザーとの紐付けが失われる。
2. **稽古とシーンの紐付け (RehearsalScene)**: シーン（Scene）が削除・再作成されるため、稽古スケジュールとの紐付けが解除される。
3. **日程調整の分析結果**: 上記の配役情報が失われることで、どの日程でどのシーンが稽古可能かの判定（シーン対応）ができなくなる。

## 解決策
脚本更新処理（`script_processor.py`）を改良し、既存の紐付け情報を一時的に退避させ、新しい脚本の解析後に名前や見出しに基づいて自動的に復元する。
